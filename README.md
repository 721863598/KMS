# ⌨️🖱️ KMS — Keyboard Mouse Statistics

> 你的每一次敲击，都值得被记录 ✨

一个轻量级的 Windows 桌面应用，**静默运行在系统托盘**，全局统计你的键盘按键和鼠标点击数据。所有数据 **100% 本地存储**，绝无联网上传 —— 你的隐私，你做主 🔒
本项目不负责保管用户数据，请用户自行妥善保管，自行备份，如有损坏不负责恢复
---

## ✨ 功能亮点

| 🎯 | 功能 |
|---|---|
| 🌍 | **全局监听** — 无论焦点在哪个窗口，都能捕获每一次按键和点击 |
| 📊 | **三维数据面板** — 按键分布、键鼠比走势、24h 点击热力，一目了然 |
| 📅 | **年度日历热力图** — 7 年跨度（2026–2032），5 阶绿色热力，每天敲了多少一目了然 |
| 🔄 | **多周期汇总** — 日 / 周 / 月 / 年，自由切换对比 |
| 🔒 | **纯本地存储** — SQLite 数据库，数据从不出你的电脑 |
| 💾 | **卸载不掉数据** — 卸载时自动备份，重装/更新自动恢复 |
| 🎨 | **无边框设计** — 毛玻璃浮窗 + 圆角卡片，颜值即正义 |
| 🪶 | **轻若无物** — 静默托盘运行，资源占用极低 |

## 🚀 怎么安装？

### 1️⃣ 下载

👉 前往 [Releases](https://github.com/721863598/KMS/releases) 下载最新的 `KMS Setup x.x.x.exe`

### 2️⃣ 安装

双击运行，一路下一步即可。安装器会自动：

- 🖥️ 创建桌面快捷方式
- 📌 创建开始菜单条目
- 📁 安装目录自动追加 `KMS` 文件夹名（避免散落文件）

### 3️⃣ 使用

软件启动后 **静默驻留在系统托盘**（右下角图标区）。**双击托盘图标** 呼出汇总面板，开始探索你的敲击数据 🎉

> 💡 首次启动会弹出隐私确认框，告诉你数据纯本地存储，放心点"确认"即可。

### ♻️ 怎么更新？

KMS 支持 **原位覆盖安装** —— 下载新版安装包，双击运行，直接覆盖旧版本即可：

| 更新时发生了什么？ | |
|---|---|
| 🔄 | 程序文件自动替换为新版本 |
| 💾 | **你的统计数据完全不受影响** — `record/stats.db` 毫发无伤 |
| 📂 | 即使卸载旧版再装新版，`record/` 也会自动备份恢复 |
| ⚡ | 全程无需手动备份，无需导入导出，傻瓜式操作 |

## 🗄️ 数据存储

```
安装目录/
├── KMS.exe          # 主程序
└── record/          # 📁 你的全部统计数据
    └── stats.db     # SQLite 数据库
```

- 📍 数据库位置：软件安装目录下的 `record/stats.db`
- 🔐 格式：SQLite，可用任何 SQLite 工具打开查看
- ♻️ 卸载时自动备份到 `KMS_Records`，下次安装自动恢复
- 🚫 **绝不联网、绝不上传、绝不分享**

## 🧡 为什么做这个？

> 当你每天敲键盘上万次、点鼠标上千次，你有没有好奇过 —— 这些动作背后藏着什么？

### 🔍 数字生活的一面镜子

我们用电脑工作、创作、娱乐、社交，键盘和鼠标是我们与数字世界之间 **唯一的物理桥梁**。每一次敲击都是一次"我在做事"的信号。但它们来去无声，从不留下痕迹。

KMS 想做的，就是 **把这座桥变成一面镜子** 🪞

### 📈 从"感觉"到"看见"

你可能觉得自己"今天打字很多"，但多到什么程度？

- 🔢 **3000 次还是 15000 次？** 数字不会骗人
- 📅 **周三比周一忙多少？** 翻翻日历热力图，一眼看穿
- ⌨️ **BackSpace 按了多少次？** 可能暴露你的犹豫和修改习惯
- 🐭 **键鼠比是多少？** 写代码的人 vs 做设计的人，比例天差地别
- 🕐 **哪个时段最专注？** 24 小时曲线就是你的一天节奏图

这些不是冷冰冰的数字，而是你 **工作习惯的体检报告** 📋

### 🛡️ 数据只属于你

市面上的统计工具要么要联网（谁在看？👀），要么要注册（凭什么？🙄），要么偷偷上传行为数据（？？？😡）。

KMS 的底线非常明确：

> **你的数据，只存在你的电脑里。没有云端，没有后台，没有遥测。**  
> 卸载时还帮你备份，下次装回来数据还在。

### 🎮 把习惯游戏化

看到日历上越来越多的绿色格子，看到自己的打字速度在进步，看到专注时段在延长 —— 这本身就是一种正反馈。KMS 不只是一个计数器，它是你数字生活的 **成绩单和升级进度条** 🏆


---

**一句话总结：KMS 是你键盘和鼠标的一生传记，秘密只属于你。** 📖🔐
